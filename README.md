# PV Module Modeling Platform (MEFLAH)

## Français
Plateforme complète de modélisation des modules photovoltaïques basée sur le modèle à deux diodes.

**Important : ce logiciel fonctionne uniquement sous Windows.**

### Fonctionnalités principales
- Tracé des courbes I-V et P-V.
- Analyse de l'influence de l'irradiance et de la température.
- Visualisation 3D du champ PV.
- Calcul des paramètres de sortie: `Pm`, `Isc`, `Voc`, `Rs`, `Rp`, `FF`, rendement.
- Export des figures et des résultats (Excel).
- Interface multilingue: Français / Arabe / English.

### Citation demandée
Si vous utilisez cette application dans un travail scientifique, merci de citer:

Meflah, A., Rahmoun, K., Mahrane, A. et al. *Outdoor performance modeling of three different silicon photovoltaic module technologies*. **Int J Energy Environ Eng** 8, 143-152 (2017).

## English
Complete PV module modeling platform based on the two-diode model.

**Important: this software works on Windows only.**

### Main features
- I-V and P-V curve plotting.
- Irradiance and temperature sensitivity analysis.
- 3D visualization of the PV array.
- Output parameter calculation: `Pm`, `Isc`, `Voc`, `Rs`, `Rp`, `FF`, efficiency.
- Figure and Excel export.
- Multilingual interface: French / Arabic / English.
